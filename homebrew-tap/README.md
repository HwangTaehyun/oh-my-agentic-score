# Homebrew Tap for Oh My Agentic Score

## Install

```bash
brew install HwangTaehyun/tap/oh-my-agentic-score
```

Or:

```bash
brew tap HwangTaehyun/tap
brew install oh-my-agentic-score
```

## Update

```bash
brew upgrade oh-my-agentic-score
```
